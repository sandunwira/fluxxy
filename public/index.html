Service Worker Example

This is an example of a service worker in action.

It will cache this page and the associated JavaScript file.

Try going offline and reloading the page.

It should still work!

Now try changing the content of the page and reloading.

It should still show the old content!

Now try changing the content of the JavaScript file and reloading.

It should still show the old content!

Now try going online and reloading the page.

It should show the new content!
